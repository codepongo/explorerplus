View/Group By ►/Group order

Current grouping target

As with Sort By's sorting target, the grouping target causes the items displayed to be sorted, but also arranged in groups.  The actual grouping method depends on the target, for example:

  • grouping by Name arranges items by the name's first letter.  Within each group (eg. all items beginning with "A"), items are also sorted by name in ascending or descending order.  Folders are grouped with files.
  • group by Size arranges items according size, namely:
    • Folders - grouped separately
    • Tiny - < 32 Kb
    • Small - < 100 Kb
    • Medium - < 1 Mb
    • Large - < 10 Mb
    • Huge - > 10 Mb

 

    Bug: When changing the order (ascending or descending) items within each group are not sorted until the Files pane is refreshed by pressing F5.  Also, grouping by Size places groups in the wrong order relative to ascending or descending.  In the current version, the Group By <file date> may not be correct in filtering files to specific groups.

 

Sort direction

Sort direction for Group By is similar to Sort By in that sorting may be done in ascending or descending order.
