Tools/Options.../Handle zip files

System Default:   No

Enabling this option allows Explorer++ to view zip files (ie. archives) as if the archive were a folder; double clicking on the file navigates to the first level of contents.

    Info: Enabling this feature over-rides any default zip handler, such as 7-Zip (an excellent freeware archive utility).  Disabling "Handle zip files" restores any external functionality.
    Incomplete: Explorer++'s author acknowledges that zip file handling is incomplete in this version.  Only the top level in any zip archive may be viewed; file operations such as Copy, etc. have not yet been implemented.
