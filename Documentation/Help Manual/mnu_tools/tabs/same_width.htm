Tools/Options.../Force all tabs to have the same width

System Default:   No

With this option enabled, Explorer++ uses tabs of all the same width, both on

  • creation of new tabs, and
  • display of existing tabs.

Use of this option frees up real estate when a large number of tabs is open, but tab labels may sometimes be truncated or obscured by the tab's icon, depending on the length of the folder's name.  The full tab address still can be viewed by hovering the mouse over the tab.
