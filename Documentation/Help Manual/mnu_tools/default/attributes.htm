Tools/Options.../Default attributes

New tabs adopt these attributes.  Each attribute may be changed on a tab-by-tab basis once the tab is created.

 

Show Hidden Files

System Default:   Yes

Enabling this tab attribute displays hidden files, subject to some system restrictions.  A more detailed description of this setting can be found here.

 

Show In Groups

System Default:   No

Enabling this tab attribute displays files and folders, grouping them by categories.  A more description of this setting can be found here.

 

AutoArrange

System Default:   Yes

Enabling this tab attribute changes the way graphical display modes arrange items in the Files pane (ie. tab).  A more description of this setting can be found here.

 

Sort Ascending

System Default:   Yes

Enabling this tab attribute changes the sorting direction for files and folders.  A more description of this setting can be found here.
