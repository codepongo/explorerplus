Tools/Options.../Use large toolbar icons

System Default:   No

Enabling this option instructs Explorer++ to switch from the default small (16x16 bit) icons to large (24x24 bit) icons.  Currently, all large icons available can be displayed when the Explorer++ is running maximized at 800x600 resolution, but future additions to the Main Toolbar may restrict this.

Large icons are only applicable to the Main Toolbar.
